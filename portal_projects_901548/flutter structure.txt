### __4. How to Modify Styling__

__To change colors:__

- Edit `lib/core/constants/app_colors.dart`
- Change hex color values (e.g., `0xFFFF6F00` for orange)

__To change overall app styling:__

- Edit `lib/core/constants/app_theme.dart`
- Modify button styles, text styles, etc.

__To change specific screen layouts:__

- Go to the specific screen file (e.g., `lib/features/auth/presentation/login_screen.dart`)
- Modify the Widget styling directly

### __5. Project Architecture__

This follows __Clean Architecture__ pattern:

- `presentation/` - UI screens and widgets
- `domain/` - Business logic and use cases
- `data/` - API calls and data sources

### __6. Development Setup__

You have both Flutter and Node.js:

- __Flutter app:__ Runs the mobile/web UI
- __Node.js proxy:__ Handles SAP API authentication and CORS
